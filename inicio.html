Dura-Line Financial Engine for Network Conduit Design

Welcome

Optical fiber network design is the first layer foundation for successful broadband and datacom deployment. Never in time has this been more important than now as the global need for future-ready infrastructure grows in importance. Optical fiber cable and conduit technology has evolved in recent years to accommodate both high bandwidth current demand and unpredictable future demands on the network. For this reason, Dura-Line developed this easy to use tool that enables the user to:

  • Select real world optical fiber and conduit network components, including labor costs, to itemize the cost of building one or more segments of a network.
  • Compare the cost for a (a) conventional construction using more traditional fiber cable, conduit, and installation techniques to (b) that using modern microcable and MicroDuct, along with potentially lower cost installation methodologies to examine the difference in total first costs.
  • Include within the cost comparison of both (a) conventional and (b) modern state-of-art network designs the cost of one or more network upgrades in specified years.
  • Calculate and obtain a comparison of the financial implications for both scenarios, including product, cost, labor, and all upgrades to provide a complete financial picture and metrics that include: NPV, IRR and ROI.

Through the use of this tool, the network design, builder, or end user can develop a quantitative based appreciation of the impact of Layer 1 Optical Fiber and Conduit network design options and the long term impact of first decisions on both, network cost and robustness.
